degreeNumberMean = p*(n-1)
print("degreeNumberMean: " + str(degreeNumberMean))

if degreeNumberMean < 1:
    print("Subcritical")
elif degreeNumberMean == 1:
    print("Critical")
elif degreeNumberMean < (n/10): # N > 1 and N < 300 (3000*0.1)
    print("Supercritical")
else: # degreeNumberMean >> N
    print("Connected")

degreeNumberMean: 2.999
Supercritical

Esta rede é supercrítica, pois possui um número médio de 2.9 arestas por nó, o que gera nós no centro com muitas conexões e nós poucos conectados não extremidades, podendo ocorrer de haver nós sem conexões, como esse número médio está próximo de 1, a rede pode vir a ser mais próxima de crítica, pois 2.9 é um valor de rede baixo para supercrítica

# c) A rede do desafio possui características de redes aleatórias? Justifique a sua resposta

A rede do desafio possui similaridade com uma rede aleatória com poucos nós e probabilidade baixa (P1N1), pois na rede aleatória de probabilidade bem baixa, o grafo com poucos nós começa apenas uma curva, mas com o incremento dos nós, vai se transformando numa curva binomial. Olhando mais a fundo a rede GraphMissingEdges percebe-se que a rede do desafio possui uma singularidade, pois ela possui por volta de 10% de seus nós conectados somente, sendo poucos nós e com graus baixos que forma a curva de grau medio conforme imagem acima (3a). As redes aleatórias apresentam suas características mais fortes no último exemplo (acima), algo bem próximo de uma curva binomial, com o grau médio bem menor que o número de nós (20 << 500,1000,2000)
